# Hospital Appointment System

A full-stack MERN Hospital Appointment System.

## Features

- Patient Authentication
- Doctor Authentication
- Appointment Booking
- Appointment Management
- Admin Dashboard
- JWT Authentication
- Responsive Design

## Tech Stack

- React
- Vite
- Node.js
- Express.js
- MongoDB Atlas
- JWT
- CSS

## Installation

Frontend

npm install
npm run dev

Backend

npm install
npm run dev
